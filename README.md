# Snake Game

A console-based Snake game built in C++ during my 1st semester Programming Fundamentals course at UET Taxila. It was one of the first real projects I built from scratch, so nothing fancy, but it works and I learned a lot making it.

## What it does

The snake moves around the console window and you have to eat the food that spawns randomly. Every time you eat, the snake gets longer and your score goes up. If you hit a wall or run into yourself, it's game over, and you get the option to play again without restarting the program.

## Features

Smooth console-based movement using keyboard input, live score tracking, collision detection for walls and the snake's own body, and a play again option at the end so you don't have to rerun it every time.

## How to run it

Open the project in any C++ compiler (I used Visual Studio), build and run, then use WASD or arrow keys to control the snake.

## Tech

Language: C++
